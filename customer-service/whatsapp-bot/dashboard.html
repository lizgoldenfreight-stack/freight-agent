⚙️ Setup Instructions

  1. Install Python dependencies: pip install pywhatkit flask
  2. Open WhatsApp Web in your browser: web.whatsapp.com
  3. Run the bot: python3 bot.py
  4. Scan QR code if prompted
  5. Bot will auto-reply to incoming messages based on keywords
  6. Customize replies in replies.json
  7. Adjust business hours in config.json
