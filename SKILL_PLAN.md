# 🛠️ Agent Skill Enhancement Plan

每个 Agent 需要安装的 ClawHub Skills（增强专业能力）

> ⚠️ 因 ClawHub 匿名速率限制，需登录后安装
> 运行 `clawhub login` 登录后，执行下方命令批量安装

---

## 一键安装全部 Skills

```bash
clawhub install chart-image --no-input
clawhub install linkedin-writer --no-input
clawhub install cold-email-writer --no-input
clawhub install invoice-tracker-pro --no-input
clawhub install weekly-report-generator --no-input
clawhub install content-writer --no-input
```

---

## 💼 Sales Support Agent — 需要的 Skills

| Skill | 用途 | 安装命令 |
|-------|------|----------|
| `chart-image` | 生成运价趋势图、margin对比图、业绩图表 | `clawhub install chart-image` |
| `linkedin-writer` | 专业LinkedIn帖子撰写（英文） | `clawhub install linkedin-writer` |
| `cold-email-writer` | Cold outreach邮件撰写，提升回复率 | `clawhub install cold-email-writer` |
| `content-writer` | 中文社媒内容生成（小红书/知乎/公众号） | `clawhub install content-writer` |

### 增强效果
- 📊 用 `chart-image` 给客户发运价趋势可视化图
- 💼 用 `linkedin-writer` 写更专业的LinkedIn帖子
- 📧 用 `cold-email-writer` 写cold email开发新客户
- 📱 用 `content-writer` 生成中文社媒内容
